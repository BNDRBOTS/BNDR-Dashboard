# BNDR — Void Glass (GitHub Pages)

## Run locally
Just open `index.html` in a browser.
(For best behavior, use a simple server: `python -m http.server`)

## Deploy to GitHub Pages
1. Create a repo (e.g., `bndr-void`)
2. Add these files:
   - `index.html`, `styles.css`, `app.js`
   - `assets/icon.svg`
3. Push to GitHub
4. Repo Settings → **Pages**
5. Source: **Deploy from a branch**
6. Branch: `main` / folder: `/root`
7. Save → wait for the Pages URL

## Data
- Stored in `localStorage` on the device.
- Use **Export** to back up, **Import** to restore.
